#  Puzzle Rearrangement Game

##  Project Overview
The Puzzle Rearrangement Game is a Python-based application that challenges users to rearrange shuffled puzzle pieces into the correct order. It is designed to improve logical thinking, problem-solving skills, and provide an engaging gaming experience.

##  Features
- Interactive puzzle gameplay
- Randomly shuffled puzzle pieces
- User-friendly interface
- Checks whether the puzzle is solved
- Easy to understand and play

##  Technologies Used
- Python 3
- Standard Python Libraries

##  How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/popuriharshitha2008-lab/Puzzle-rearrangement-game.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Puzzle-rearrangement-game
   ```

3. Run the program:
   ```bash
   python puzzle_rearrangement_game.py
   ```
